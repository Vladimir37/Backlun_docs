Chat Backlun

Main page - Backlun/front/chat/index.html.

Static folder - Backlun/front/chat/static/.

You can get the file from static folder at the address /src.

Example: Backlun/front/chat/static/js/build/main.js -> /src/js/build/main.js

Easy example a chat client:

const client = new require('websocket').client(); const ws_client = new client(); ws_client.on('connect', function(connection) { if (connection.connected) { // Send message to the chat connection.sendUTF(JSON.stringify({message: "some text"})); } }

note: nickname will be set automatically!

Test it: tests/test.chat.ws.js

Chat API

Method Address Params Response Description
ws /api/ws - Open a websocket connection Open websocket connection and write/read some msg

WebSocket API

Name Type Data Description
connect Server - WebSocket Client Connected
connectFailed Server - Connect error
close Server - Connection closed
message Server utf8Data string Message received
