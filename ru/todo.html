Вы можете получить статичные файлы из каталога по адресу /src.

Команда для запуска тестов: npm install && npm run test

ToDo API

Параметры, отмеченные астериском (*) обязательны. Содержимое поля "Ответ" расположено в поле "body" ответа сервера. При запросах, измеяющих существующие объекты, необходимо отправлять данные всех полей объектов - неотправленные поля будут очищены.

Метод Адрес Параметры запроса Ответ Описание
GET /api/tasks/get_all/ - Массив Объектов задач Получить массив всех задач
GET /api/tasks/get_category/ id* int
 Массив Объектов задач Получить массив всех задач в выбранной категории
POST /api/tasks/add/ title* string
text* string
category int 		Объект задачи Создать новую задачу
POST /api/tasks/edit/ id* int
title string
text string
category int
status bool 		Объект задачи Изменить задачу
POST /api/tasks/delete/ id* int - Удалить задачу
GET /api/categories/get_all/ - Массив Объектов категорий Получить массив всех категорий
POST /api/categories/add/ name* string
color* string 		Объект категории Создать новую категорию
POST /api/categories/edit/ id* int
name string
color string 		Объект категории Изменить категорию
POST /api/categories/delete/ id* int - Удалить категорию

Стандартные объекты

Ответ сервера:

status int
message string
body any

Код статуса: 0 - успешно, 1-9 - различные ошибки, 10 - не найдено.

Объект задачи:

ID int
Title string
Text string
Category int
Status bool

Поле "category" должно содержать ID существующей категории или 0.

Объект категории:

ID int
Name string
Color string

Вы не можете удалить категорию, пока в ней есть задачи.
