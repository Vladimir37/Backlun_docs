Backlun

Описание

Backlun — это набор различных бэкенд-сервисов (платформ) с API, но без фронтенда. Вы можете создать любой фронтенд с помощью любых библиотек, фреймворков или JS-диалектов. Backlun не требует установки, не имеет зависимостей и работает на Linux, Windows и OS X. Достаточно скачать Backlun и начать использовать его.

Зачем?

Backlun будет полезен для фронтенд-разработчиков, не владеющих навыками разработки на бэкенде, но желающих потренироваться в создании фронтенда или дизайне для настоящих динамических приложений. Backlun подходит для разработчиков, которые хотят испытать в деле новый фреймворк, но не хотят писать бэкенд приложение для этого. Backlun станет хорошим выбором для нанимателя, который хочет проверить навыки нанимаемого фронтенд-разработчика.

Примеры использования

Вы хотите научиться писать фронтенд на TypeScript или ELM. Вы прочитали туториал по React и хотите создать первое настоящее приложение. Вы хотите нанять нового фронтенд-разработчика, но он ранее работал только с вёрсткой статичных страниц и вы хотите проверить его способности по созданию приложений, взаимодействующих с сервером через API. Вам нужен простой инструмент для тестовых заданий для кандидатов на вакансию.

Платформы

  • ToDo List — Записывайте свои цели и задачи, разбивайте их по категориям и выполняйте.
  • Blog — Ведите свой блог, создавайте новые посты, добавляйте к ним теги и оставляйте комментарии.
  • Market — Кладите товар в корзину, пополняйте счёт виртуальными деньгами и совершайте покупки.
  • Forum — Регистрируйтесь, общайтесь на форуме, создавайте новые темы и отвечайте в них.
  • Blackjack — Сыграйте в классическую карточную игру и попытайтесь набрать 21 очко раньше ваших соперников.
  • Calendar — Ведите ежедневник, отмечайте в нём прошедшие и грядущие события, как быстрые, так и растянутые во времени.
  • Geopos — Выберите точку на карте и получите расстояние между отмеченным местом и вами.
  • OAuth — Используйте новейший OAuth 2.0 сервис авторизации.
  • Chat — Общайтесь в чате на вебсокетах.

Установка

Скачивание

Выберите желаемую версию на странице релизов или выберите её ниже:

  • Backlun Windows
  • Backlun Linux
  • Backlun OS X

Скачайте архив и распакуйте его в любое удобное вам место.

Запуск приложения

  • Windows — Запустите Backlun.exe внутри каталога.
  • Linux and OSX — Выполните Backlun через терминал. Пример команды: ./Backlun. Если это необходимо, дайте файлу права на исполнение: chmod +x Backlun.

Запуск платформ

После запуска приложения вы увидите терминал. Выполните одну из следующих команд:

  • help — Показать полную помощь
  • help <platform> — Показать помощь по выбранной платформе.
  • start <platform> [<port>] — Запустить выбранную платформу. Вы можете дополнительно указать порт.

Примеры:

  • start blog — Запустить платформу "blog" на порте 8000.
  • start forum 5000 — Запустить платформу "forum" на порте 5000.
  • help market — Показать помощь по платформе "market".
Примечание для платформы "OAuth"

Запуск платформы "OAuth" отличается от запуска других платформ.

start oauth [<port> <host> <key>]

key - ключевой файл. По умолчанию имя файла - key.json и он должен быть расположен в корневом каталоге Backlun/. Изначально этот файл отсутствует и должен быть создан пользователем при необходимости. Структура ключевого файла:

{ "cid": string, "csecret": string }

Компиляция

Вы можете не использовать готовые бинарные файлы, а скомпилировать Backlun самостоятельно. Скачайте исходный код Backlun. Распакуйте в любое удобное место, перейдите в директорию и выполните go build Backlun.go. Если у вас не установлен Golang, вы можете сделать это по иснтрукции. Для успешной компиляции у вас должны быть установлены все зависимости. Вы можете увидеть недостающие при попытке начать компиляцию.

Тесты

Также для каждой платформы существуют тесты. Вы можете найти тесты для каждой платформы в каталоге tests/. Вы можете использовать код тестов для написания фронтенда.
